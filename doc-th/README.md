# HerbReady — คู่มือการใช้งานภาษาไทย

ยินดีต้อนรับสู่ **HerbReady** — ระบบจ่ายยาสมุนไพรสำหรับโรงพยาบาลและคลินิก

เอกสารฉบับนี้จะแนะนำการใช้งานโปรแกรมตั้งแต่การติดตั้ง การตั้งค่า ไปจนถึงการใช้งานฟีเจอร์ต่างๆ อย่างละเอียด

## เริ่มต้นใช้งาน

หากคุณเพิ่งดาวน์โหลดโปรแกรมนี้ไป แนะนำให้อ่านตามลำดับดังนี้:

1. **อ่านภาพรวม** — เข้าใจว่าโปรแกรมนี้ทำอะไรได้บ้าง
2. **ติดตั้งโปรแกรม** — ทำตามขั้นตอนการติดตั้ง
3. **เชื่อมต่อฐานข้อมูล** — เชื่อมต่อกับฐานข้อมูลของโรงพยาบาล
4. **ตั้งค่าโปรแกรม** — กำหนดค่ายา ห้องตรวจ และกฎต่างๆ
5. **ใช้งาน** — เริ่มใช้งานฟีเจอร์ต่างๆ

## ข้อกำหนดเบื้องต้น

ก่อนใช้งาน HerbReady คุณต้องมี:

- **ฐานข้อมูล MySQL 8.0+** — เป็นฐานข้อมูลของระบบ HIS ของโรงพยาบาล (เช่น HosXP, PMQA, ฯลฯ)
- **Node.js 18+** — สำหรับ development
- **Rust 1.70+** — สำหรับ build backend
- **ระบบปฏิบัติการ** — Windows, macOS หรือ Linux
