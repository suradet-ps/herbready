# 5. คู่มือการใช้งาน

HerbReady มี 3 แท็บหลักสำหรับใช้งาน:

1. **แท็บคิววันนี้** — โหลดและประมวลผลคิวผู้ป่วยประจำวัน
2. **แท็บค้นหาผู้ป่วย** — ค้นหาผู้ป่วยรายบุคคล
3. **แท็บประวัติการจ่ายยา** — ดูประวัติการจ่ายยาสมุนไพรย้อนหลัง

และมี **แผงเลือกยา (Drug Panel)** ที่ด้านขวาของหน้าจอสำหรับเลือกยาที่จ่าย

---

## การนำทาง (Navigation)

- คลิกที่แท็บด้านบนเพื่อสลับระหว่างหน้าต่างๆ
- แถบด้านล่างแสดงสถานะการเชื่อมต่อฐานข้อมูล
- ไอคอนรูปเฟือง (⚙️) มุมขวาบนสำหรับเปิดการตั้งค่า
