# 5-1. แท็บคิววันนี้ (Daily Queue)

แท็บ "คิววันนี้" ใช้สำหรับโหลดและประมวลผลรายชื่อผู้ป่วยที่มาพบแพทย์แผนไทยในวันที่เลือก

---

## การโหลดคิวประจำวัน

### ขั้นตอนที่ 1: เลือกวันที่

1. คลิกที่ช่อง **"วันที่"** หรือ **"Date"**
2. เลือกวันที่ที่ต้องการ (ค่าเริ่มต้นเป็นวันปัจจุบัน)

### ขั้นตอนที่ 2: เลือกห้องตรวจ

1. คลิกที่ช่อง **"ห้องตรวจ"** หรือ **"Department"**
2. เลือกห้องตรวจที่ต้องการ (ต้องมีการตั้งค่าห้องตรวจก่อน ดูที่ [การตั้งค่า](./4-configuration.md))

### ขั้นตอนที่ 3: โหลดคิว

1. คลิกปุ่ม **"โหลดคิว"** หรือ **"Load Queue"**
2. รอสักครู่ โปรแกรมจะดึงข้อมูลผู้ป่วยจากฐานข้อมูล
3. เมื่อโหลดเสร็จ จะแสดงรายชื่อผู้ป่วยในตารางด้านล่าง

---

## การอ่านข้อมูลในตารางคิว

ตารางคิวจะแสดงข้อมูลผู้ป่วยดังนี้:

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| **VN** | หมายเลขการมาพบแพทย์ (Visit Number) |
| **HN** | หมายเลขผู้ป่วย (Hospital Number) |
| **ชื่อ-นามสกุล** | ชื่อและนามสกุลผู้ป่วย |
| **ห้องตรวจ** | ห้องตรวจที่ผู้ป่วยมาพบ |
| **สิทธิ์** | ประเภทสิทธิ์การรักษา |
| **วันที่มาล่าสุด** | วันที่มาพบแพทย์ครั้งล่าสุด |
| **น้ำหนัก** | น้ำหนักตัวล่าสุด (กก.) |
| **ความดัน** | ความดันโลหิต (mmHg) |
| **ชีพจร** | อัตราการเต้นของหัวใจ (bpm) |
| **ยาที่จ่ายได้** | รายชื่อยาสมุนไพรที่พร้อมจ่าย |
| **ไม่เคยจ่าย** | รายชื่อยาสมุนไพรที่ไม่เคยจ่ายให้ผู้ป่วย |
| **ยังไม่ถึงกำหนด** | รายชื่อยาที่ต้องรออีกกี่วัน |
| **เลือก** | ช่องสำหรับเลือกผู้ป่วยเพื่อพิมพ์/ส่งออก |

---

## กลุ่มยาสมุนไพร

### 1. ยาที่จ่ายได้ (Eligible)

หมายถึงผู้ป่วยที่ห่างจากการรับยาสมุนไพรชนิดนั้นเกินกว่าระยะเวลาคอร์สที่กำหนดแล้ว สามารถจ่ายยาได้ทันที

**ตัวอย่าง:**
```
ฟ้าทะลายโจร (30 วัน)
→ หากผู้ป่วยรับยาครั้งล่าสุดเมื่อ 35 วันก่อน → จ่ายได้
```

### 2. ไม่เคยจ่าย (Never Dispensed)

หมายถึงผู้ป่วยที่ไม่เคยรับยาสมุนไพรชนิดนี้เลย ต้องตัดสินใจว่าจะจ่ายหรือไม่

**ตัวอย่าง:**
```
มะขามแขก → ผู้ป่วยไม่เคยรับยานี้มาก่อน
```

### 3. ยังไม่ถึงกำหนด (Not Yet Eligible)

หมายถึงผู้ป่วยที่เพิ่งรับยาไป และยังอยู่ในระยะเวลาคอร์ส ต้องรออีกกี่วันถึงจะจ่ายได้

**ตัวอย่าง:**
```
ขิง (14 วัน)
→ หากผู้ป่วยรับยาเมื่อ 5 วันก่อน → ต้องรออีก 9 วัน
```

---

## การดูรายละเอียดผู้ป่วย

คลิกที่แถวของผู้ป่วยในตารางเพื่อดูรายละเอียดเพิ่มเติม:

- **ข้อมูลส่วนตัว:** ชื่อ, HN, CID, สิทธิ์การรักษา
- **ข้อมูลการมาพบ:** วันที่, ห้องตรวจ, VN
- **สัญญาณชีพ:** น้ำหนัก, ความดัน, ชีพจร
- **ประวัติยา:** รายชื่อยาสมุนไพรแต่ละชนิดและสถานะ
- **ผลแลป:** ผลแลปล่าสุดที่มีการตั้งค่า
- **ปฏิกิริยาระหว่างยา:** แจ้งเตือนยาแผนปัจจุบันที่อาจมีปฏิกิริยากับยาสมุนไพร

---

## การกรองและค้นหา

### กรองตามสถานะยา

สามารถกรองผู้ป่วยที่มียาสมุนไพรในสถานะต่างๆ:

- **แสดงทั้งหมด** — แสดงผู้ป่วยทุกคน
- **มียาจ่ายได้** — แสดงเฉพาะผู้ป่วยที่มียาที่จ่ายได้
- **มียาไม่เคยจ่าย** — แสดงเฉพาะผู้ป่วยที่มียาไม่เคยจ่าย
- **มียายังไม่ถึงกำหนด** — แสดงเฉพาะผู้ป่วยที่มียายังไม่ถึงกำหนด

### ค้นหาผู้ป่วย

ใช้ช่องค้นหาเพื่อกรองผู้ป่วยตาม:
- ชื่อ-นามสกุล
- HN
- CID

พิมพ์ข้อมูลที่ต้องการค้นหา รายการจะถูกกรองอัตโนมัติ

---

## การแจ้งเตือน

### แจ้งเตือนปฏิกิริยาระหว่างยา

เมื่อผู้ป่วยมีการใช้ยาแผนปัจจุบันที่อาจมีปฏิกิริยากับยาสมุนไพร จะมี **ไอคอนประกาศ (!)** แสดงที่แถวของผู้ป่วย

คลิกเพื่อดูรายละเอียด:
- ชื่อยาแผนปัจจุบันที่มีปฏิกิริยา
- ชื่อยาสมุนไพรที่เกี่ยวข้อง
- เหตุผล/คำเตือน

### แจ้งเตือนผลแลปผิดปกติ

เมื่อผลแลปล่าสุดของผู้ป่วยมีค่าผิดปกติตามกฎที่ตั้งไว้ จะมี **ไอคอนแลป (🧪)** แสดงที่แถวของผู้ป่วย

คลิกเพื่อดู:
- รายการแลปที่ผิดปกติ
- ค่าที่วัดได้ vs ค่าขีดจำกัด

---

## การเลือกผู้ป่วยเพื่อพิมพ์/ส่งออก

1. ที่คอลัมน์ **"เลือก"** ในแต่ละแถว ให้คลิกช่องเลือก
2. หรือใช้ปุ่ม **"เลือกทั้งหมด"** / **"ยกเลิกเลือกทั้งหมด"**

ผู้ป่วยที่ถูกเลือกจะถูกนำไปใช้ในการ:
- พิมพ์ใบสั่งยา (PDF)
- ส่งออกเป็น Excel

---

## การรีเฟรชข้อมูล

คลิกปุ่ม **"รีเฟรช"** หรือกดปุ่ม **F5** เพื่อโหลดข้อมูลคิวใหม่อีกครั้ง
