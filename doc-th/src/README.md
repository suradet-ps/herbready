# HerbReady - คู่มือการใช้งาน

ยินดีต้อนรับสู่ **HerbReady** — ระบบจ่ายยาสมุนไพรสำหรับโรงพยาบาลและคลินิก

เอกสารฉบับนี้จะแนะนำการใช้งานโปรแกรมตั้งแต่การติดตั้ง การตั้งค่า ไปจนถึงการใช้งานฟีเจอร์ต่างๆ อย่างละเอียด

---

## เริ่มต้นใช้งาน

หากคุณเพิ่งดาวน์โหลดโปรแกรมนี้ไป แนะนำให้อ่านตามลำดับดังนี้:

1. **[ภาพรวมของโปรแกรม](1-overview.md)** — เข้าใจว่าโปรแกรมนี้ทำอะไรได้บ้าง
2. **[การติดตั้งโปรแกรม](2-installation.md)** — ทำตามขั้นตอนการติดตั้ง
3. **[การเชื่อมต่อฐานข้อมูล](3-database-connection.md)** — เชื่อมต่อกับฐานข้อมูลของโรงพยาบาล
4. **[การตั้งค่าโปรแกรม](4-configuration.md)** — กำหนดค่ายา ห้องตรวจ และกฎต่างๆ
5. **[คู่มือการใช้งาน](5-user-guide.md)** — เริ่มใช้งานฟีเจอร์ต่างๆ

---

## ฟีเจอร์หลัก

- **ระบบคิวประจำวัน** — โหลดและประมวลผลคิวผู้ป่วยประจำวัน พร้อมจัดกลุ่มยาตามสถานะ
- **ระบบค้นหาผู้ป่วย** — ค้นหาด้วย HN, CID หรือชื่อ-นามสกุล
- **ระบบประวัติการจ่ายยา** — ดูประวัติการจ่ายยาสมุนไพรย้อนหลัง
- **ระบบแจ้งเตือน** — แจ้งเตือนปฏิกิริยาระหว่างยาและผลแลปผิดปกติ
- **ระบบพิมพ์และส่งออก** — พิมพ์ใบสั่งยา PDF และส่งออก Excel

---

## ข้อกำหนดเบื้องต้น

ก่อนใช้งาน HerbReady คุณต้องมี:

- **ฐานข้อมูล MySQL 8.0+** — เป็นฐานข้อมูลของระบบ HIS ของโรงพยาบาล
- **Node.js 18+** — สำหรับ development
- **Rust 1.70+** — สำหรับ build backend
- **ระบบปฏิบัติการ** — Windows, macOS หรือ Linux
